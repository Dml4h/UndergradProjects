## A collection of Parallel Processing projects for CSCI 6330-Parallel Processing
### Project Descriptions
   - Project 1: A sequential C program to solve the hotplate problem
   - Project 2: C program using pthreads to solve the hotplate problem
   - Project 3: C program that using pthreads to examine a search space looking for one or more solutions. 
   - Project 4: MPI C program to retrieve and display each rank's hostname & hostid
   - Project 5: MPI C program to solve the hotplate problem
   - Project 6: CUDA C program that performs as a single layer of neurons in a neural network and produces 1-D vector output
